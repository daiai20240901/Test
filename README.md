# iPhone App Starter Guide

このリポジトリは、iPhoneアプリを作るための最小ガイドとSwiftUIのサンプルを提供します。

## 進め方の概要
1. **目的を決める**: どんな課題を解決するアプリか、機能を箇条書きで整理します。
2. **画面設計**: 画面の一覧と各画面の役割を決めます。
3. **データ設計**: 画面で扱うデータ（モデル）を定義します。
4. **実装**: SwiftUIでUIを作り、必要ならAPIやローカル保存を追加します。
5. **テスト**: Xcodeのプレビューやシミュレーターで動作確認します。

## 次に決めると良いこと
- ターゲットOS（例: iOS 17）
- ログインが必要か
- オフライン対応が必要か
- バックエンド（API）が必要か

## サンプル
`ios/MyApp` に、最小のSwiftUI構成例を置いています。
- `MyAppApp.swift`: アプリのエントリーポイント
- `ContentView.swift`: 画面のサンプル

> 注意: これはXcodeプロジェクトではありません。Xcodeで新規プロジェクトを作成し、
> 同名ファイルを置き換えて試す想定です。
